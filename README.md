# The-View
 美容室のフリマサイトです。<br>
 使わなくなった物を他の美容室の誰かにあげたり、探している物を無料でもらえたり出来ます。<br>
 レスポンシブ対応しているのでスマホからもご確認いただけます。
 
# URL
http: <br>
画面上部のゲストログインボタンから、メールアドレスとパスワードを入力せずにログインできます。
